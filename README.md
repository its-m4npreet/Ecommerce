# Ecommerce
A full-featured Ecommerce website built with React and Node.js, designed for scalability, performance, and a seamless shopping experience.

## üöÄ Features

- **User Authentication:** Secure registration, login, and logout with JWT.
- **Product Catalog:** Browse, filter, and search products with dynamic listings.
- **Shopping Cart & Checkout:** Add to cart, manage quantities, and complete orders with Stripe payment integration.
- **Order Management:** Track order history and status for users and admins.
- **Admin Dashboard:** Manage products, categories, users, and orders with intuitive controls.
- **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
- **RESTful API:** Robust backend with Express and MongoDB.
- **Error Handling & Validation:** Comprehensive feedback for user actions.

## üì¶ Getting Started

### Prerequisites

- Node.js & npm
- MongoDB

### Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/ecommerce.git
    ```
2. **Install dependencies:**
    ```bash
    cd ecommerce
    npm install
    cd client
    npm install
    ```
3. **Configure environment variables:**  
   Create `.env` files in both backend and frontend directories with your settings (MongoDB URI, JWT secret, Stripe keys, etc.).

4. **Start development servers:**
    ```bash
    # Backend
    npm run server

    # Frontend
    cd client
    npm start
    ```
